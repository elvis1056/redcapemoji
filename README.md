# Elvis 專案

這是一個使用 React 建立的個人前端專案

> [Elvis 個人網站](http://www.redcapemoji.com/)

## 使用技術和第三方套件

- Create React App
- React Hook (^17.0.2)
- ESLint + Prettier 檢查冗余程式碼 + 統一代碼風格
- AWS Ubuntu / Nginx 部署網站
- [React Router](https://www.npmjs.com/package/react-router) - Web App 路由管理
